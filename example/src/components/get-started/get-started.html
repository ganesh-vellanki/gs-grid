Get started

Prerequisites
  • Node.js 14+ and Yarn 1.x
  • A browser with Web Components support
Run locally
  1. Install dependencies
  2. Build source and docs
  3. Start demo server
yarn install
yarn start
Install in your app
yarn add @gs-grid/gs-grid
Add grid element
<gs-grid></gs-grid>
Configure columns and data
import { GridConfig, GridEvents } from '@gs-grid/gs-grid/dist/model';

const gridConfig = new GridConfig();
gridConfig.rowHeight = 32;
gridConfig.columnDefs = [
  { field: 'name', headerName: 'Name' },
  { field: 'email', headerName: 'Email' },
  { field: 'company.name', headerName: 'Company', width: 180 }
];
gridConfig.data = data;

const gridEl = document.querySelector('gs-grid');
GridEvents.setupGridConfig(gridEl.getAttribute('instance-id'), gridConfig);
Useful scripts
yarn build
yarn dev
yarn clean
